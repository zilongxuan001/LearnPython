# 171019数据类型-字符串

字符串，就是在单引号`''`、双引号  `""`、三引号 `'''  '''`   `""" """`

里的数字或字母，或其他符号。

举例

`'a'`

`"123"`

`'&%'`

```
"""
good moring
my wife
this is Monday
"""
```
